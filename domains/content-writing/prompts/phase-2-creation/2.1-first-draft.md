# Prompt: Content Writer - First Draft Creation

**Persona:** Content Writer
**Phase:** 2 - Creation
**Purpose:** Transform content outline into engaging first draft with brand voice and natural keyword usage

---

## When to Use This Prompt

✅ **Use when:** Content brief and outline are approved, ready to write
❌ **Don't use when:** Brief is incomplete or outline needs refinement

---

## The Prompt Template

```markdown
You are a Content Writer. Your task is to create an engaging first draft following the approved outline.

**Rules:**
1. You must adhere to all principles in the attached `CONSTITUTION.md` file (brand voice, tone, style).
2. The draft must be in Markdown format.
3. The output must be saved as `artifacts/02-draft-v1.md`
4. The draft must include:
   - Compelling title (H1) with primary keyword
   - Strong hook in first 100 words
   - All sections from outline (H2/H3 structure)
   - Natural keyword usage (NOT stuffing)
   - Short paragraphs (2-4 sentences, <150 words)
   - Bullet lists and formatting for scannability
   - Clear CTA at end
   - Meta description (150-160 characters)
5. **CRITICAL:** Write for humans first, search engines second. Prioritize clarity and engagement over keyword density.

**Context:**
- Target word count: [WORD_COUNT] words
- Target audience: [TARGET_AUDIENCE]
- Brand voice: [BRAND_VOICE] (from constitution)
- Primary keyword: [PRIMARY_KEYWORD]

**Files to Read:**
- `CONSTITUTION.md` (for brand voice and tone)
- `artifacts/01-content-brief.md` (for topic, audience, goals)
- `artifacts/01-content-outline.md` (for structure)

---

Here is what I want you to write:

[TOPIC_DESCRIPTION]

---

**Additional Context:**
- Tone: [TONE] (helpful, conversational, professional, etc.)
- Examples to include: [EXAMPLES]
- CTA: [CALL_TO_ACTION]
```

---

## Quick Start Guide

### Step 1: Prepare
- [ ] Content brief approved
- [ ] Outline approved
- [ ] Brand voice from constitution understood

### Step 2: Fill Variables
- Replace `[WORD_COUNT]` → e.g., "1,800"
- Replace `[TARGET_AUDIENCE]` → Specific audience
- Replace `[BRAND_VOICE]` → e.g., "Helpful, conversational"
- Replace `[PRIMARY_KEYWORD]` → Main keyword

### Step 3: Invoke AI & Write
Writer creates first draft following outline

### Step 4: Review Draft
Check:
- [ ] Hook is compelling (first 100 words)
- [ ] Brand voice is consistent
- [ ] Keywords used naturally
- [ ] Paragraphs are short
- [ ] CTA is clear

---

## Tips for Better Drafts

✅ **DO:**
- Start with strong hook (stat, question, bold statement)
- Keep paragraphs short (2-4 sentences)
- Use bullet lists and formatting
- Write in second person ("you")
- End with clear CTA

❌ **DON'T:**
- Keyword stuff
- Write wall-of-text paragraphs
- Use generic introductions
- Forget the CTA
- Ignore brand voice
